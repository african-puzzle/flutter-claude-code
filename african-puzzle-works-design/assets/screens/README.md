## Export Settings

- **Format**: PNG
- **Scale**: 2x (for retina displays)
- **Background**: Include (for screens), Transparent (for components)
- **File Naming**: Use lowercase with hyphens (e.g., `bottom-sheets.png`)

## How to Export from Figma

1. Select the frame in Figma
2. In the right panel, scroll to **Export** section
3. Click **+** to add export settings
4. Choose PNG, 2x scale
5. Click **Export [frame name]**
6. Save to this directory

## Purpose

These screenshots serve as visual reference for developers implementing the design system, helping ensure pixel-perfect implementation and providing quick visual lookup for component specifications.
